## Описание проекта

Проект направлен на разработку предсказательных моделей для анализа эффективности игроков NBA на основе открытых статистических данных. Планируется провести сбор и очистку данных, исследовательский анализ, визуализацию ключевых метрик и построение рейтингов игроков.

Основное внимание уделяется применению методов машинного обучения и анализа временных рядов для решения практических задач:
- прогнозирование результата матча;
- оценка трансферной стоимости игрока;
- выявление перспективных спортсменов;
- кластеризация игровых профилей.

Отдельное направление исследования — сравнение классических ML-подходов с моделями глубокого обучения (включая RNN и LSTM) с целью повышения точности прогнозов и оценки применимости различных архитектур для спортивной аналитики.
